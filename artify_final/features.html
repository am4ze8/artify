About Artify

Key features implemented in this student project.

Gallery (Upload & Preview)

Upload images or audio. Files are converted to base64 with FileReader.readAsDataURL() and saved in localStorage. Preview items in the modal.

Client-side Search

Live search by title, author or description — runs completely in the browser using the saved posts.

Contact Form (Demo)

Form performs basic validation and stores messages in localStorage so you can demo persistence during viva.

Responsive Design

Layout adapts to mobile and desktop with clear spacing and readable typography.

Frontend-only

No backend required for the assignment — everything stored in browser for demo purposes.

Examiner-friendly

Starter content is seeded so demo always shows items. Code has descriptive function names for easy explanation.
